Instructions

1. If you are from the VPAA office, perform a Banner query for the following information for the faculty body: ACAD_YEAR, NAME_ALPHS, DIV, RANK, NETWORK_USERNAME.

2. Open the results in Excel.

3. Add a column titled 'RETIREMENT/LEAVE/SABBATICAL' and enter the data for the appropriate faculty members. This information is PUBLIC, meaning when any faculty member searches for information about this person, the data entered in the 'RETIREMENT/LEAVE/SABBATICAL' column will be visible.



4. Once the retirement/leave/sabbatical information has been added, save the Banner query Excel file as a csv file (comma-separated values) that is titled FacultyInformation.csv.
